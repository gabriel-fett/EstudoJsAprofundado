# Dia 1 — var, let, const

## O que aprendi

const é para valores que não mudam
let é para valores que podem mudar
var é o antigo e vaza do bloco

## O que me confundiu no começo

Achei que "true" e true eram a mesma coisa

## Regra que vou lembrar

Aspas só para texto. Números e booleans sem aspas.
